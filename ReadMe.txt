Как мною пользоваться:

1. запускаем Main.py ( для этого нужна библиотека pywin32 и python)
2. запускаем камеру на компьютере и ставим в удобное место так, чтобы была видна расцветка
3. когда программа запросит поставить курсор на индикатор ставим наш курсор чуть ниже/левее/правее/выше индикатора, чтобы цвет четко распозновался 
!!! Обрати внимание - программа обрабатывает только 3 цвета, а ёще программа не будет виновата, если камера сфокуситруется неправильно(зеленый будет похож на синий, например) !!!
4. нажимаем enter для ком. строки
5. готово! Можно закрепить камеру, чтобы её не загораживали другие приложения

!!!Внимание (2) - всю папку нужно хранить в том состоянии, какая она на гите, иначе работать не будет, а также в коде по необходимости исправить расположение самой папки(base) - по умолчанию на раб. столе !!!
